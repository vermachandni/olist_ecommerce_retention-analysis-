
# Olist E-Commerce: Churn & Logistics Analysis 🚚📈

## 📌 Project Overview
Olist was facing a **15% drop in customer retention** and rising logistics costs. This project builds a Python pipeline to identify why customers leave and predicts future churn.

## 🛠️ Tools Used
- **Python:** Pandas, NumPy (Data Cleaning & Merging)
- **Visualization:** Matplotlib, Seaborn (Exploratory Data Analysis)
- **Machine Learning:** Scikit-Learn (Random Forest Churn Prediction)

## 💡 Key Insights
- **The "Delivery Gap":** 1-star reviews are directly correlated with an average delivery time of 19 days, compared to 10 days for 5-star reviews.
- **Top Churn Driver:** The predictive model identified `actual_delivery_days` as the #1 factor in customer churn.
- **Retention Rate:** Identified a low repeat-purchase rate of X% (mention your calculated result).
